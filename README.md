

## Features Documentation

### Cart System

The cart system provides:
- Add/remove items
- Quantity management
- Color selection
- Bulk discounts (10% off for 5+ items)
- LocalStorage persistence
- Animated feedback
- Cart dropdown

### Product Quick View

The product modal shows:
- Product images
- Available colors
- Stock status
- Bulk discount info
- Add to cart functionality
- Quantity selection

### Animations

- Toast notifications
- Cart icon bounce
- Modal fade-in
- Product hover effects
- Smooth transitions
